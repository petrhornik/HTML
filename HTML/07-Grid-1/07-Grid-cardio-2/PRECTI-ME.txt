Grid cardio 1 a 2 jsou stejné příklady, jako jsme měli 
v lekcích o flexboxu.

Vyzkoušej si, jak se stejné rozložení dělá pomocí
CSS gridu. Zjistíš, že některé se řeší gridem mnohem
jendoduššeji než flexboxem. To bude platit hlavně pro
příklady ve složce Flexbox-cardio-2. U některých příkladů
to vyjde nastejno a u jiných vyhraje na množství kódu
flexbox.

I když se jedná o stejné příklady, které už jsi jednou
dělal/a, opravdu ti doporučuji si je zkusit.

Tyto příklady ti mají pomoci procvičit si základy
vytváření CSS gridu i ukázat, že Grid není následník
flexboxu, na který teď můžeme zapomenout, ale že tyto
technologie existují v CSS současně a doplňují se.

A my jako autoři CSS kódu se radujeme, protože máme
k dispozici obě a můžeme je používat, jak se nám zlíbí.
