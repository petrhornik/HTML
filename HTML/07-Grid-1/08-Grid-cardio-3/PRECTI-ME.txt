Pamatuješ na tyto příklady z lekcí flexboxu?

A jak byly někdy komplikované a musel/a jsi vymýšlet,
jak do sebe vše zabalit, aby to šlo nakódovat pomocí flexboxu?

Tak schválně, zkus si je pomocí CSS gridu.
Uvidíš, jak extrémně jednoduché to najednou je.

CSS grid jako nástroj pro layout stránky je naprosto BOŽÍÍÍÍÍ.

Což neznamená, že flexbox je k ničemu. Má stále svoje použití
a používáme ho často. Ale na tyto příklady je CSS Grid
naprosto nepřekonatelný.
